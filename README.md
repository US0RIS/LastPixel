# LastPixel

A collaborative pixel art canvas inspired by r/place, where creativity never stops.

## Features

- **Interactive Canvas**: 1024×1024 pixel board with zoom and pan
- **Credit System**: Prepaid pixel placement with dynamic pricing
- **Weekly Resets**: Fresh canvas every 7 days with automatic archiving
- **Undo System**: 5-minute window to undo placements (with escalating costs)
- **Leaderboards**: Track top contributors
- **Archives**: Browse past weekly canvases
- **Voting System**: Monthly community voting with credit rewards
- **Reporting**: Community moderation with automatic board freeze at threshold

## Tech Stack

- **Backend**: Python FastAPI
- **Database**: SQLite
- **Frontend**: Vanilla HTML/CSS/JavaScript with Canvas API

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the server:
```bash
python main.py
```

3. Open browser to `http://localhost:5000`

## Deployment

Designed to run on Railway, Render, or any Python hosting platform.
